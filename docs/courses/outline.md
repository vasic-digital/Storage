# Course: Object Storage in Go

Learn to build storage-agnostic applications using `digital.vasic.storage` -- from local files to S3 and multi-cloud.

## Lessons

1. **Core Interfaces and Local Storage** -- The `ObjectStore` and `BucketManager` interfaces, functional options, and the local filesystem client.
2. **S3-Compatible Storage** -- Connecting to MinIO/AWS S3, presigned URLs, bucket lifecycle rules, and configuration.
3. **Cloud Providers and Storage Routing** -- Managing multi-cloud credentials with `CloudProvider`, and routing assets to backends with the `Resolver`.
